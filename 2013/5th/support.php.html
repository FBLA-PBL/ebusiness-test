› About Us:
This website was created for the 2012-2013 FBLA E-Business Prompt; in particular, Lynbrook Grad was created for the purpose of managing seat reservation and ticket sales for the Class of 2013 and their family members.

Lynbrook Grad is supported on multiple browsers, including recent versions of Google Chrome, Mozilla Firefox, Internet Explorer, Safari, and Opera. This website has also been optimized for mobile devices that run on iOS, Android, Windows Phone, and Blackberry OS.

We are a team of hardworking, dedicated students who had a vision to create this website. Each of us specialized in a different field of expertise for the project.
